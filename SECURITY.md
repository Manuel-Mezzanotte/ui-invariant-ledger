# Security Policy

UI Invariant Ledger is a markdown-first Agent Skill. It does not execute code by default.

## Reporting Security Issues

Please report security concerns through GitHub security advisories or a private report if available.

Do not use public issues for sensitive reports.

## Scope

Security-sensitive reports may include:

- malicious instructions hidden in skill content;
- unsafe install or publish instructions;
- supply-chain concerns around releases, tags, or repository configuration;
- documentation that could cause users to overtrust unverified behavior.

## Out Of Scope

UI Invariant Ledger is not a security scanner. It can flag high-risk UI surfaces such as auth, payments, destructive actions, permissions, and user data, but it does not replace a security review.
